# Question Prioritization

Use this framework to ask fewer, higher-value follow-ups.

## Prioritization Rule
Rank questions by:
1. Decision impact
2. Risk reduction
3. Irreversibility
4. Cost of delay

## Question Categories
- Goal alignment:
  - What outcome matters most.
- Scope boundaries:
  - What is explicitly out of scope.
- Constraints:
  - Time, platform, policy, or compatibility limits.
- Acceptance:
  - How success will be evaluated.

## Output Format
- Top open questions (max 5) with:
  - Why this question matters
  - Owner
  - Due-by
  - Default assumption if unanswered

## Stop Condition
- Continue gathering only while answers materially change plan direction.
- If remaining unknowns are low impact, proceed with explicit assumptions.
